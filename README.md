# Receipt Template for Small Business

Standalone MVP project for the keyword/opportunity `receipt template for small business`.

## Goal
Ship an independent landing-page + lightweight receipt-template tool to validate:
- CTA clicks
- template preference
- editing intent
- print/export completion

## MVP scope
- 1 landing page
- 3 template directions: Classic, Modern Clean, Compact
- editable receipt fields
- live preview
- print first, export next

## Project status
- independent local project directory created
- standalone Git repo initialization in progress
- GitHub repo creation in progress
- deployment target to be linked after repo baseline is ready
